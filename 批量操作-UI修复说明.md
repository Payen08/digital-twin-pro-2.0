# 批量操作 UI 修复说明

## ✅ 已修复的问题

### 1. 位置、旋转、缩放只显示一个输入框
**问题**: 之前使用 `PropRow` 包裹，所有3个输入框在一行，看起来像只有一个
**修复**: 改为每个轴单独一行显示

#### 修复前
```jsx
<PropRow label="位置">
    <div className="flex gap-1 w-full">
        <SmartInput /> {/* X */}
        <SmartInput /> {/* Y */}
        <SmartInput /> {/* Z */}
    </div>
</PropRow>
```

#### 修复后
```jsx
<div className="space-y-2">
    <div className="flex items-center gap-2">
        <label className="text-[11px] text-gray-500 w-12 text-right">位置 X</label>
        <SmartInput />
    </div>
    <div className="flex items-center gap-2">
        <label className="text-[11px] text-gray-500 w-12 text-right">位置 Y</label>
        <SmartInput />
    </div>
    <div className="flex items-center gap-2">
        <label className="text-[11px] text-gray-500 w-12 text-right">位置 Z</label>
        <SmartInput />
    </div>
</div>
```

### 2. JSX 标签不匹配错误
**问题**: `PropRow` 改为 `div` 后，关闭标签没有同步修改
**修复**: 将所有 `</PropRow>` 改为 `</div>`

## 💡 改进点

1. **清晰的标签** - 每个输入框都有明确的标签（位置 X、位置 Y、位置 Z）
2. **垂直布局** - 每个轴单独一行，更容易阅读和操作
3. **统一样式** - 所有输入框使用相同的布局和样式
4. **右对齐标签** - 标签右对齐，输入框左对齐，视觉更整齐

## 🔧 技术细节

### 布局结构
```jsx
<PropSection title="位置">
    <div className="space-y-2">  {/* 垂直间距 */}
        <div className="flex items-center gap-2">  {/* 每行 */}
            <label className="text-[11px] text-gray-500 w-12 text-right">
                位置 X
            </label>
            <SmartInput value={...} onChange={...} />
        </div>
        {/* Y 和 Z 同样结构 */}
    </div>
</PropSection>
```

### 样式说明
- `space-y-2` - 行间距 0.5rem
- `flex items-center gap-2` - 水平布局，垂直居中，间距 0.5rem
- `w-12` - 标签宽度 3rem
- `text-right` - 标签右对齐
- `text-[11px]` - 字体大小 11px
- `text-gray-500` - 灰色文字
