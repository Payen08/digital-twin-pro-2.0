# 🚀 批量操作功能 - 快速使用指南

## ✅ 集成完成！

批量操作功能已经成功集成到你的应用中。现在你可以立即使用！

## 🎯 如何使用（3步）

### 1️⃣ 添加一些对象
在场景中添加几个对象（墙体、立方体、柱子等）

### 2️⃣ 框选对象
- **按住 Shift 键**
- **拖动鼠标**画出选择框
- 释放鼠标完成选择

### 3️⃣ 执行操作
选中后，右上角会显示操作面板，点击相应按钮即可

## 🎨 操作面板功能

```
┌─────────────────────────────────┐
│  已选择 3 个对象                 │
├─────────────────────────────────┤
│  🗑️ 删除    │  📋 复制          │
│  📦 组合    │  📏 缩放          │
│  🔄 旋转    │  📐 对齐          │
│  ↔️ 水平分布 │  ↕️ 垂直分布      │
│  ❌ 取消选择                     │
└─────────────────────────────────┘
```

## 💡 使用技巧

### 框选技巧
- ✅ **必须按住 Shift 键**才能框选
- ✅ 框选时会显示**蓝色虚线框**
- ✅ 可以框选**任意数量**的对象

### 操作技巧
- 🗑️ **删除**：会弹出确认对话框
- 📋 **复制**：对象会在 X 轴偏移 2 个单位
- 📦 **组合**：多个对象会合并成一个 Group
- 📏 **缩放**：输入比例（如 1.5 = 放大 1.5 倍）
- 🔄 **旋转**：输入角度（如 45 = 旋转 45 度）
- 📐 **对齐**：输入方式（left/center/right/top/bottom）
- ↔️ **水平分布**：需要至少 3 个对象
- ↕️ **垂直分布**：需要至少 3 个对象

## ❓ 常见问题

### Q: 按 Shift 拖动没反应？
**A:** 确保：
- ✅ 场景中有可选择的对象
- ✅ 对象是可见的（visible: true）
- ✅ 对象没有被锁定（locked: false）

### Q: 面板不显示？
**A:** 检查：
- ✅ 是否成功框选到对象
- ✅ 浏览器控制台是否有错误
- ✅ 刷新页面重试

### Q: 操作没效果？
**A:** 确认：
- ✅ 对象有正确的 geometry 和 material
- ✅ 查看控制台日志
- ✅ 尝试其他操作
