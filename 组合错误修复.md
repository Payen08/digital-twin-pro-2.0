# 组合功能错误修复

## 错误
```
Cannot read properties of undefined (reading 'count')
at new EdgesGeometry
```

## 原因
组对象 `type: 'group'` 不应该被渲染，但渲染循环没有过滤它。

## 修复
在两个地方添加过滤：

### 1. 2D渲染 (行784)
```javascript
objects.filter(obj => obj.visible && obj.type !== 'group')
```

### 2. 3D渲染 (行4728)
```javascript
displayObjects.filter(obj => 
    !obj.isBaseMap && 
    obj.type !== 'map_image' && 
    obj.type !== 'waypoint' && 
    obj.type !== 'path_line' && 
    obj.type !== 'group'  // 添加这行
)
```

## 测试
1. 框选2个对象
2. 点击"组合"
3. 验证：无错误，对象可见

已修复！
