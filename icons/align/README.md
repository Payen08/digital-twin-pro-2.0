# 对齐图标文件夹

请将以下图标 PNG 文件放在这个文件夹中：

## 需要的图标文件

1. **align-left.png** - 左对齐图标
2. **align-center.png** - 居中对齐图标
3. **align-right.png** - 右对齐图标
4. **distribute-horizontal.png** - 水平分布图标
5. **distribute-vertical.png** - 垂直分布图标

## 图标规格

- **尺寸**: 24x24 像素（推荐）
- **格式**: PNG（支持透明背景）
- **颜色**: 白色或浅灰色（适配深色背景）
- **风格**: 简洁、扁平化

## 图标示例说明

### align-left.png
```
|  □
|  □
|  □
```
三个方块左边缘对齐

### align-center.png
```
  □
  □
  □
```
三个方块居中对齐

### align-right.png
```
  □  |
  □  |
  □  |
```
三个方块右边缘对齐

### distribute-horizontal.png
```
□  □  □
```
三个方块水平均匀分布

### distribute-vertical.png
```
□
□
□
```
三个方块垂直均匀分布

## 使用方法

将图标文件放在此文件夹后，应用会自动加载并显示。

如果没有图标文件，会显示默认的 SVG 图标。
